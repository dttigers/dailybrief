## Logo

- **Mark:** Diamond eye + V stroke
- **Lockup:** Icon + "Vigil" wordmark (Inter 500)
- **Variants:** On dark, on light, on teal
- **Icon sizes:** 64px, 48px, 32px, 20px, 14px
- **Clear space:** Height of the V stroke around the mark
- **Minimum size:** 14px icon / 80px wordmark

### Don'ts

- Do not stretch or distort
- Do not use off-brand colors
- Do not place on busy backgrounds
- Do not add drop shadows or effects

## Typography

### Typeface — Inter

Primary typeface across all surfaces — web, app, print. Open source, zero licensing cost.

### Type Scale

| Level | Usage |
|-------|-------|
| H1    | Page titles, hero headline |
| H2    | Section headings |
| H3    | Card headings, labels |
| Body  | Thought content, dashboard text |
| Small | Captions, tags, filter pills |
| Micro | Timestamps, metadata |

### Weights

Two weights only:

- **400 Regular** — body text, captions, UI elements
- **500 Medium** — headings, labels, button text
- Never use 600 Bold or 700 Heavy — too heavy against the UI

## UI Components

### Tag System — Shape Encodes Meaning

- **Rounded pill** = category (what kind of thought): Therapy, Task, Idea, Reflection, Project
- **Square badge** = routing (where it goes): Bring to therapist, Self-work

### Buttons

- **Primary:** Teal 600 fill, white text
- **Secondary:** Outline, no fill

### Thought Row (Dark Theme)

- Tags + timestamp on first line
- Thought content below
- Subtle separator between rows

## Voice & Tone

### Brand Personality

- **Calm:** Never urgent. Never alarming. The product holds things so the user doesn't have to feel anxious.
- **Confident:** Direct and clear. No hedging. If Vigil says it captured something, it did.
- **Empathetic:** Built for ADHD brains. Knows that friction is the enemy. Meets the user where they are.
- **Quiet:** The product should feel like it's working in the background. It doesn't demand attention.

### Writing Guidelines

**Do:**
- Use plain, direct language
- Speak to the user's relief, not their problem
- Short sentences — ADHD brains appreciate brevity
- First person for the product: "Vigil captured this"

**Don't:**
- Don't use productivity jargon (synergy, leverage, etc.)
- Don't be clinical or cold — this holds personal thoughts
- Don't over-explain — trust the user
- Don't use ALL CAPS for emphasis
