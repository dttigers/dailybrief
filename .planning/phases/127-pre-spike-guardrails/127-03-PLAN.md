<objective>
Ship the GUARD-02 server-side primitives: a PCM byte-cap helper module + Vitest entry in PWA ERROR_CODE_MAP. The 60-second cap is enforced as a base64-length check before any decoding — defense-in-depth at HTTP ingress.
