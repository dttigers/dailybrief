7. When file is skipped (already in manifest): `NSLog("[FolderWatcher] Skipping already-processed: %@", filename)`

Use NSLog (not print) so messages appear in Console.app. Follow existing logging patterns in the file. Keep logging concise — one line per event, not verbose.
