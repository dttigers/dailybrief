<task type="auto">
  <name>Task 1: Create DashboardViewModel, DashboardView, and ThoughtRowView</name>
  <files>Sources/DailyBriefMonitor/Dashboard/DashboardViewModel.swift, Sources/DailyBriefMonitor/Dashboard/DashboardView.swift, Sources/DailyBriefMonitor/Dashboard/ThoughtRowView.swift</files>
  <action>
  Create a `Dashboard/` subdirectory under `Sources/DailyBriefMonitor/`.

  **DashboardViewModel.swift:**
  - `@Observable class DashboardViewModel` with `@MainActor`
  - Properties: `thoughts: [Thought]`, `searchQuery: String`, `selectedCategory: ThoughtCategory?`, `isLoading: Bool`, `totalCount: Int`, category counts dict
  - Takes `ThoughtStore` in init
  - `loadThoughts()` async method — calls `store.fetchAll(category:)` or `store.search(query:)` depending on state
  - `loadCounts()` async method — calls `store.count()` and `store.count(category:)` for each category to populate sidebar badges
  - Debounced search: when `searchQuery` changes, wait 300ms then reload. Use a Task-based debounce pattern (cancel previous task, sleep, then fetch).
  - `refresh()` async method that reloads both thoughts and counts

  **DashboardView.swift:**
  - NavigationSplitView with sidebar + detail
  - Sidebar: "All" row with total count badge, then one row per ThoughtCategory with count badge. Highlight selected. Use ThoughtCategory.displayColor for sidebar icons (circle.fill SF Symbol).
  - Detail: VStack with search bar (TextField with magnifyingglass icon) at top, then List of ThoughtRowView for each thought. Show "No entries" empty state when list is empty. Show "No results" when search yields nothing.
  - Pull the ThoughtCategory UI extension (displayColor, displayName) from CaptureView.swift into a shared file OR just reference it — it's already in CaptureView.swift and both files are in the same target, so it's accessible.
  - `.task` modifier calls `viewModel.refresh()` on appear
  - `.onChange(of: viewModel.selectedCategory)` triggers reload
  - `.searchable` modifier on the detail column bound to `viewModel.searchQuery`

  **ThoughtRowView.swift:**
  - Takes a `Thought` binding/value
  - Shows: content text (2-line truncation), category pill (using ThoughtCategory.displayColor/displayName), confidence %, relative timestamp (e.g., "2 hours ago")
  - Use `RelativeDateTimeFormatter` for timestamps
  - Layout: VStack with content on top, HStack with category pill + confidence + timestamp on bottom
  </action>
  <verify>swift build compiles without errors</verify>
  <done>DashboardViewModel fetches and filters thoughts, DashboardView renders list with sidebar and search, ThoughtRowView shows individual entries</done>
</task>
