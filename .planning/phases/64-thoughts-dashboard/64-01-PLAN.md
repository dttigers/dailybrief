## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-64-01 | Tampering | ThoughtRow content display | mitigate | React JSX auto-escapes all interpolated strings — do NOT use dangerouslySetInnerHTML. Content is rendered via `{thought.content}` only. |
| T-64-02 | Tampering | SearchBar input | mitigate | Search value passed as `q` query param — URLSearchParams auto-encodes special characters. Server uses parameterized `plainto_tsquery`, immune to SQL injection. |
| T-64-03 | Denial of Service | ThoughtList rendering | accept | Max 50 items per fetch; layout uses `line-clamp-3` to prevent content overflow. Acceptable risk for single-user app. |
</threat_model>
