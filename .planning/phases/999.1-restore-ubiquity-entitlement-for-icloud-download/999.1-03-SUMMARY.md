## Decisions Made

- Used literal `iCloud.com.jamesonmorrill.dailybriefmonitor` (not `$(TeamIdentifierPrefix)`) — codesign invoked from shell does not expand Xcode build-setting macros; macro would be embedded verbatim and fail profile matching
- Bare-binary entitlements file uses empty dict (Pitfall 6 Option 1) rather than omitting entitlements entirely — codesign requires an explicit entitlements path; empty dict is the correct "no entitlements" signal
- Edit 4 in install.sh anchored by finding `--identifier "com.jamesonmorrill.dailybriefmonitor"` in the no-`--deep` codesign block, then changing only the immediately-following `--entitlements` line — prevents naive global substitution from mutating the .app codesign --deep block at line 208
- Cross-machine constraint: Linux host lacks Swift toolchain, so per-task `swift build -c release` gates were deferred to a single Mac-side verification after Task 4 push (and before writing this SUMMARY)
