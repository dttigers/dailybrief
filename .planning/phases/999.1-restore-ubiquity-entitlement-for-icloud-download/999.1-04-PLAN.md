## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| dry-run branch ↔ $INSTALL_DIR | install.sh writes into `~/.local/bin/` — same path used by main. A regression here affects the operator's daily-driver Vigil install. |
| codesign ↔ amfid | First-launch validation. Failure mode is process termination, not file corruption — recovery is reinstall, not data restore. |
