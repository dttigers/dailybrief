<task type="auto">
  <name>Task 2: Create TherapyPatternService and TherapyPrepService actors</name>
  <files>Sources/JarvisCore/Services/TherapyPatternService.swift, Sources/JarvisCore/Services/TherapyPrepService.swift</files>
  <action>
  **TherapyPatternService.swift:**
  - Create `TherapyPatternError` enum (apiError, parseError) following InsightError pattern
  - Create `TherapyPatternService` actor with `init(apiKey:model:)` following InsightService pattern
  - Add `detectPatterns(thoughts:lookbackDays:)` async throws method returning `[TherapyPattern]`
  - Require minimum 5 therapy thoughts to detect patterns (return empty otherwise)
  - System prompt should:
    - Identify recurring emotional themes, behavioral patterns, and unresolved concerns across therapy thoughts
    - Note which themes are increasing, stable, or decreasing in frequency
    - Focus on genuine patterns, not surface-level observations
    - Emphasize: "You are a pattern detection tool, NOT a therapist. Surface observations to help the user prepare for therapy sessions."
  - User message: format thoughts as `[id] (classification, date) content` (same as InsightService)
  - Request JSON array response: `[{"theme": "...", "description": "...", "frequency": N, "trend": "increasing|stable|decreasing", "related_thought_ids": [], "confidence": 0.0-1.0}]`
  - Use max_tokens: 1024
  - Filter results to confidence >= 0.5
  - Parse response following InsightService double-parse pattern (outer Claude response → inner JSON)

  **TherapyPrepService.swift:**
  - Create `TherapyPrepError` enum (apiError, parseError)
  - Create `TherapyPrepService` actor with `init(apiKey:model:)`
  - Add `generatePrep(thoughts:patterns:)` async throws method returning `TherapyPrep`
    - `thoughts` = recent bringToTherapist thoughts
    - `patterns` = optional [TherapyPattern] from pattern service (can be empty)
  - Require minimum 1 thought (return empty TherapyPrep otherwise)
  - System prompt should:
    - Generate a structured therapy session prep from the user's recent thoughts marked for therapist discussion
    - If patterns provided, use them to add context about recurring themes
    - Organize topics by urgency (high/medium/low)
    - Suggest a session focus based on what seems most pressing
    - Emphasize: "You are organizing the user's own thoughts for their therapy prep, NOT providing therapy or clinical advice."
  - User message: format thoughts + optional patterns context
  - Request JSON response: `{"items": [{"topic": "...", "context": "...", "urgency": "high|medium|low", "related_thought_ids": []}], "overall_themes": ["..."], "suggested_focus": "..."}`
  - Use max_tokens: 1024
  - Parse response following same double-parse pattern
  </action>
