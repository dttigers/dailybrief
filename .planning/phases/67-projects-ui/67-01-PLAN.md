Projects API (vigil-core/src/routes/projects.ts):
```
GET  /v1/projects          -> ProjectApiResponse[]
GET  /v1/projects/:id      -> ProjectApiResponse
POST /v1/projects          -> ProjectApiResponse (201)
PATCH /v1/projects/:id     -> ProjectApiResponse
DELETE /v1/projects/:id    -> 204

ProjectApiResponse = { id: number, name: string, description: string | null, status: string | null, createdAt: string, updatedAt: string }
```

Thoughts API project filtering (vigil-core/src/routes/thoughts.ts):
```
GET /v1/thoughts?projectId=N      -> PaginatedResponse<ThoughtApiResponse>  (thoughts in project N)
GET /v1/thoughts?unassigned=true  -> PaginatedResponse<ThoughtApiResponse>  (thoughts with no project)
PUT /v1/thoughts/:id  body: { projectId: N }    -> assign to project
PUT /v1/thoughts/:id  body: { projectId: null }  -> unassign from project
```
