    1. Create `src/routes/prioritize.ts`:
       - POST /prioritize accepting JSON body:
         ```
         {
           workOrders: Array<{
             caseNumber: string,
             store: string,
             shortDescription: string,
             trade: string,
             location: string,
             equipment: string,
             priority: string,
             contact: string,
             state: string
           }>
         }
         ```
       - Validate: at least 1 work order required, return 400 if empty
       - If AI client unavailable, return 503
       - Implement daily cache with content hash (matching Swift behavior):
         - Cache dir: `~/.cache/dailybrief/`
         - Generate hash from sorted JSON of work order case numbers (use crypto.createHash("md5"))
         - Cache file: `wo-priority-YYYY-MM-DD-{hash}.json`
         - If cache exists, return cached result
       - Call `callClaude()` with:
         - system prompt (port from Swift WorkOrderPrioritizer):
           ```
           You are a facilities management assistant. Analyze these work orders and rank them by urgency. Consider: safety hazards (electrical, water, gas), customer/business impact (HVAC in extreme weather, security issues, food safety), time-sensitivity (perishable equipment, active leaks), and trade complexity.
           Respond with ONLY a JSON array of case numbers in priority order (highest urgency first), e.g. ["CS0353601", "CS0353598"]. No other text.
           ```
         - userMessage: numbered list of work orders, each formatted as:
           ```
           N. Case: {caseNumber}
              Store: {store}
              Description: {shortDescription}
              Trade: {trade}
              Location: {location}
              Equipment: {equipment}
              Priority: {priority}
              Contact: {contact}
              State: {state}
           ```
         - maxTokens: 500
       - Parse JSON response as string[]
       - Write to cache file
       - Return 200 with `{ prioritizedCaseNumbers: string[], cached: boolean }`
    2. Mount in `src/index.ts`

           ```
           ]
           ```
         - maxTokens: 300
