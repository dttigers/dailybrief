<task type="auto">
  <name>Task 2: Create GoogleCalendarService actor</name>
  <files>Sources/DailyBrief/Services/GoogleCalendarService.swift</files>
  <action>
  Create GoogleCalendarService.swift as an actor in Sources/DailyBrief/Services/ following the existing SportsService pattern (actor with config init, URLSession for HTTP, JSON parsing).

  The service needs:

  1. `actor GoogleCalendarService` initialized with `GoogleCalendarConfig`
  2. Private method `refreshTokenIfNeeded() async throws -> String` that:
     - Loads CalendarTokens from file
     - If not expired, returns accessToken
     - If expired, POSTs to `https://oauth2.googleapis.com/token` with grant_type=refresh_token, refresh_token, client_id, client_secret
     - Parses response for new access_token and expires_in
     - Updates CalendarTokens (keep existing refreshToken, update accessToken and expiresAt)
     - Saves updated tokens
     - Returns new accessToken

  3. `func fetchTodayEvents() async throws -> [CalendarEvent]` that:
     - Calls refreshTokenIfNeeded() to get valid token
     - For each calendarId in config.selectedCalendarIds (or ["primary"] if empty):
       - GET `https://www.googleapis.com/calendar/v3/calendars/{calendarId}/events`
       - Query params: timeMin (start of today ISO8601), timeMax (end of today ISO8601), singleEvents=true, orderBy=startTime, maxResults=50
       - Authorization: Bearer {accessToken}
     - Parse response JSON: items array -> CalendarEvent structs
       - Map `summary` to title, `start.dateTime` or `start.date` to startTime, `end.dateTime` or `end.date` to endTime, `location` to location
       - If `start.date` exists (no dateTime), mark isAllDay=true
       - Use ISO8601DateFormatter for dateTime parsing
       - Set calendarName from the calendar's summary (or calendarId)
     - Combine events from all calendars, sort by startTime
     - Return sorted [CalendarEvent]

  4. `func fetchCalendarList() async throws -> [(id: String, name: String)]` that:
     - GET `https://www.googleapis.com/calendar/v3/users/me/calendarList`
     - Returns array of (id, summary) tuples for the settings UI calendar picker
     - Used during OAuth setup to let user select which calendars to include

  Error handling: throw descriptive errors. If tokens don't exist (not yet authorized), throw a clear "Not authorized" error. Caller (Generate command) will catch and skip gracefully.

  Follow existing patterns: URLSession.shared.data(for:), JSONSerialization for parsing (not Codable for API responses — matches TriageService/SportsService approach), actor for thread safety.
  </action>
  <verify>swift build compiles without errors. Service initializes with config.</verify>
  <done>GoogleCalendarService actor compiles. fetchTodayEvents() and fetchCalendarList() methods implemented. Token refresh logic complete.</done>
</task>
