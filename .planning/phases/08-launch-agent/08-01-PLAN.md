<task type="auto">
  <name>Task 1: Create install.sh script</name>
  <files>Scripts/install.sh</files>
  <action>
Create a shell script that:
1. Builds both targets in release mode: `swift build -c release`
2. Creates `~/.local/bin/` if it doesn't exist
3. Copies DailyBrief CLI binary from `.build/release/DailyBrief` to `~/.local/bin/DailyBrief`
4. Copies DailyBriefMonitor binary from `.build/release/DailyBriefMonitor` to `~/.local/bin/DailyBriefMonitor`
5. Generates LaunchAgent plist for the monitor at `~/Library/LaunchAgents/com.jamesonmorrill.dailybriefmonitor.plist` using the installed path (`~/.local/bin/DailyBriefMonitor`). Include RunAtLoad=true, KeepAlive=true, StandardOutPath and StandardErrorPath to `~/Library/Logs/DailyBrief/`.
6. Creates the log directory `~/Library/Logs/DailyBrief/` if it doesn't exist.
7. Loads/reloads the monitor LaunchAgent: unload existing if present, then `launchctl load`.
8. Prints summary of what was installed.

The script should be idempotent (safe to run multiple times). Use `set -euo pipefail` for safety. Make it executable with `chmod +x`.

Do NOT install the CLI LaunchAgent (the 6 AM scheduler) — that will be handled by the monitor internally in plan 08-02.
  </action>
  <verify>bash Scripts/install.sh --help or reading the script confirms it handles all steps. chmod +x is set.</verify>
  <done>Scripts/install.sh exists, is executable, and handles build + install + LaunchAgent setup for the monitor</done>
</task>
