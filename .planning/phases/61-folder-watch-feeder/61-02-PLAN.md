**Part A: AppDelegate lifecycle wiring**

Modify `Sources/DailyBriefMonitor/AppDelegate.swift`:

1. Add a new stored property near the other service properties (around line 20):
```swift
private(set) var folderWatcher: FolderWatcherService?
```

The `private(set)` allows `DailyBriefMonitorApp` to read it for state observation.

2. In `applicationDidFinishLaunching`, after the `captureService` is created (after line 68, inside the `do` block), create and start the watcher:

```swift
// Folder watcher — headless file feeder (Phase 61)
if let imgService = self.imageDescriptionService as? APIImageDescriptionService {
    let watcher = FolderWatcherService(
        imageService: imgService,
        transcriptionService: transcription,
        captureService: service
    )
    self.folderWatcher = watcher
    Task { await watcher.start() }
} else {
    NSLog("DailyBriefMonitor: folder watcher skipped — image service not available")
}
```

Key detail: `self.imageDescriptionService` is typed as `(any ImageDescriptionProviding)?` but is actually an `APIImageDescriptionService`. Cast it to the concrete type because `FolderWatcherService` needs the concrete actor (it calls `processPhoto(imageURL:preview:forcePaperType:)` which is on the concrete type, not the protocol).

3. In `applicationWillTerminate` (line 140), add watcher shutdown before the hotkey unregister:

```swift
func applicationWillTerminate(_ notification: Notification) {
    if let watcher = folderWatcher {
        // Synchronous bridge — stop must cancel DispatchSources and close fds
        let semaphore = DispatchSemaphore(value: 0)
        Task {
            await watcher.stop()
            semaphore.signal()
        }
        semaphore.wait(timeout: .now() + 2)  // 2s max wait
    }
    globalHotKey?.unregister()
}
```
