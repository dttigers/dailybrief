<!-- Backend brief history API (from vigil-core/src/routes/brief-history.ts) -->

GET /v1/briefs — List briefs
  Query params: limit (default 30, max 200), offset (default 0), from (YYYY-MM-DD), to (YYYY-MM-DD)
  Response: { data: BriefApiResponse[], total: number, limit: number, offset: number }

GET /v1/briefs/:date — Get single brief by date (YYYY-MM-DD)
  Response: BriefApiResponse
  404 if not found

BriefApiResponse shape:
```typescript
{
  id: number;
  date: string;         // YYYY-MM-DD
  summary: unknown;     // JSON object — the brief's structured content
  pdfFilename: string | null;
  thoughtCount: number;
  taskCount: number;
  createdAt: string;    // ISO 8601
}
```
