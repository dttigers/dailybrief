# Phase 129.1 Plan 03: POST /v1/captures/screenshot Summary

**One-liner:** New `POST /v1/captures/screenshot` route on vigil-core ingests a base64 PNG/JPG screenshot of a Polaris case page, runs Anthropic vision (claude-sonnet-4-20250514 via `callClaudeMultimodal()` with per-user spend tracking from Phase 127 GUARD-03), parses the JSON output, and writes a `work_orders` row in `state: "pending_review"` for operator confirmation in the PWA — preserving SVCNOW-04 SELECT-by-(userId, clientCaptureId) dedup short-circuit and handling case_number PK collisions via `onConflictDoUpdate` (latest screenshot wins; operator's review step is the safety net). Operator decision locked Task 1 upfront: `department` maps to the Polaris "Maintenance Location" field.

## What Was Built

### Task 2 — Route file + mount (commit `060d111`)

**`vigil-core/src/routes/captures-screenshot.ts` (337 LOC):**

- **DI factory** `createCapturesScreenshotRoute(deps: Partial<CapturesScreenshotDeps> = {})` returns a Hono router. Defaults: real `db`, real `getAIClient`, real `callClaudeMultimodal`. Tests override the three via deps. Mirrors `createWorkOrdersRoute(deps)` at `work-orders.ts:58`.
- **Pre-wired singleton** `export const capturesScreenshot = createCapturesScreenshotRoute()` at module bottom — mounted by `index.ts`.
- **Constants** (all exported for tests): `VALID_MEDIA_TYPES = ["image/png", "image/jpeg"] as const`, `UUID_V4_REGEX` (verbatim from RESEARCH Q6), `MAX_IMAGE_BASE64_LENGTH = 7_000_000`, `EXTRACTION_PROMPT` (operator-decided "Maintenance Location" mapping).
- **Type interfaces** (all exported): `RequiredExtracted` (5 string fields), `ExtraExtracted` (13 optional string fields), `CapturesScreenshotDeps`.
- **Handler flow** (POST `/captures/screenshot`):
  1. `userId = c.get("userId")` — populated by `bearerAuth` dispatcher (Phase 121 D-D2 lock; T-129.1-09 spoofing mitigation).
  2. 503 if `!dbAvailable`.
  3. JSON parse (400 on parse failure).
  4. Validate `imageBase64` (non-empty string → 400; length > MAX_IMAGE_BASE64_LENGTH → 413; T-129.1-10 DoS guard).
  5. Validate `mediaType` ∈ {image/png, image/jpeg} (400).
  6. Validate `clientCaptureId` is non-empty + UUID v4 shape (400; T-129.1-13 replay mitigation).
  7. 503 if `!getAIClientFn()`.
  8. SVCNOW-04 dedup: SELECT-by-(userId, clientCaptureId). Hit → 200 with `duplicate: true`, echoed `extractedFields`, and `extraExtractedFields` parsed from the stored `notes` JSON. No AI call on dedup hit.
  9. `callClaudeMultimodalFn({content: [imageBlock, textBlock], maxTokens: 1500, userId})` — AI throw → 502.
  10. `parseAIJson<ExtractionResult>(rawText)` — handles fence-wrapped JSON.
  11. Validate `extracted.required.case_number` non-empty (422).
  12. `INSERT into workOrders ... ON CONFLICT (case_number) DO UPDATE SET ...` — state="pending_review", maintenanceProblem + department populated from extracted.required, notes=JSON.stringify(extras). Latest screenshot wins per RESEARCH Risk 1 / T-129.1-14 accepted risk.
  13. Return 200 with `{workOrderId, duplicate: false, extractedFields, extraExtractedFields}`.
