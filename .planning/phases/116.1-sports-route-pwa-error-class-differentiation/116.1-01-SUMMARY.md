## Decisions Made

- **UpstreamError.cause**: carries original thrown error (network/DNS/socket) for ops diagnostics, NOT serialized to HTTP responses — Plan 02 reads `.kind` and `.retryAfter` only
- **Retry-After sanitization**: `parseInt(raw, 10)` + `Number.isFinite(parsed) && parsed > 0 && parsed <= 86_400` — prevents malformed BDL headers (e.g. XSS injection strings) reaching downstream consumers
- **_timeoutMsOverride**: test seam pattern (injectable dep) rather than mocking timers — avoids fake-timer complexity; Plan 02 can pass `_timeoutMsOverride: 10` for a "hanging fetch" test
- **T-73-01**: the `console.log(BDL fetch failed: ...)` line is preserved as the ONLY place provider name + URL appear in failure paths
