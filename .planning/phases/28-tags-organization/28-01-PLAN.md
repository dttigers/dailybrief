         createdAt DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
         UNIQUE(sourceThoughtId, targetThoughtId)
       )

     - Follow existing model patterns (Codable + FetchableRecord + PersistableRecord).

  NOTE: tags uses [String]? not [String] so existing rows with NULL tags column don't need backfill. When decoding NULL, GRDB returns nil. The UI will treat nil and [] the same (no tags shown).

<task type="auto">
  <name>Task 2: ThoughtStore methods for tags, favorites, and links</name>
  <files>Sources/JarvisCore/Storage/ThoughtStore.swift</files>
  <action>
  Add the following methods to ThoughtStore actor, following existing patterns (db.reader.read for queries, write for mutations, set modifiedAt + syncStatus on mutations):

  **Tag methods:**
  - `func addTag(id: Int64, tag: String) throws -> Thought?` — Fetch thought, append tag to tags array (creating array if nil), dedup, save. Return updated thought.
  - `func removeTag(id: Int64, tag: String) throws -> Thought?` — Fetch thought, remove tag from array, save. Return updated thought.
  - `func fetchByTag(tag: String, limit: Int = 100, offset: Int = 0) -> [Thought]` — Query thoughts WHERE tags LIKE '%"tag"%'. Use SQL LIKE since JSON array contains quoted strings.
  - `func allUniqueTags() -> [String]` — Fetch all non-null tags columns, decode each JSON array, flatten, deduplicate, sort alphabetically. Return unique tag list.
  - `func bulkAddTag(ids: Set<Int64>, tag: String) throws -> Int` — Add tag to multiple thoughts in single transaction. Return count modified.
  - `func bulkRemoveTag(ids: Set<Int64>, tag: String) throws -> Int` — Remove tag from multiple thoughts in transaction.

  **Favorite methods:**
  - `func toggleFavorite(id: Int64) throws -> Thought?` — Fetch thought, flip isFavorited, save. Return updated.
  - `func fetchFavorites(limit: Int = 100, offset: Int = 0) -> [Thought]` — WHERE isFavorited = 1 ORDER BY modifiedAt DESC.
  - `func countFavorites() -> Int` — COUNT WHERE isFavorited = 1.

  **Link methods:**
  - `func linkThoughts(sourceId: Int64, targetId: Int64) throws -> ThoughtLink?` — INSERT OR IGNORE into thought_links. Also insert reverse link (targetId, sourceId) for bidirectional. Return the link.
  - `func unlinkThoughts(sourceId: Int64, targetId: Int64) throws` — DELETE both directions from thought_links.
  - `func fetchLinkedThoughts(thoughtId: Int64) -> [Thought]` — JOIN thought_links on either sourceThoughtId or targetThoughtId matching thoughtId, return the OTHER thought in each pair. Deduplicate.
  - `func countLinks(thoughtId: Int64) -> Int` — Count linked thoughts for a given thought.

  **Update fetchFiltered:**
  - Add optional `tag: String?` parameter to existing `fetchFiltered` method. When non-nil, add WHERE tags LIKE '%"tag"%' condition.
  - Add optional `favoritesOnly: Bool = false` parameter. When true, add WHERE isFavorited = 1.
  - Update `countFiltered` similarly.

  All mutation methods must set `modifiedAt = Date()` and `syncStatus = .pending` on affected Thought rows (not on thought_links — links are local-only, no sync).
  </action>
  <verify>swift build succeeds. Verify all new public methods are accessible from outside the module.</verify>
  <done>ThoughtStore has complete CRUD for tags, favorites, and links. All mutations mark thoughts as pending sync. fetchFiltered supports tag and favorites filtering.</done>
</task>
