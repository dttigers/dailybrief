```typescript
export interface TherapyPattern {
  theme: string; description: string; frequency: number;
  trend: 'increasing' | 'stable' | 'decreasing';
  relatedThoughtIds: number[]; confidence: number;
}

export interface TherapyPrepItem {
  topic: string; context: string; urgency: 'high' | 'medium' | 'low';
  relatedThoughtIds: number[];
}

export interface TherapyPrep {
  items: TherapyPrepItem[]; overallThemes: string[]; suggestedFocus: string;
}

export async function getThoughts(params: { limit?: number; ... }): Promise<ThoughtsListResponse>
export async function getTherapyPatterns(thoughts, days?): Promise<{ patterns: TherapyPattern[] }>
export async function generateTherapyPrep(thoughts, patterns?): Promise<TherapyPrep>
```

Backend constraints:
- /v1/therapy/patterns requires minimum 5 thoughts with therapyClassification
- /v1/therapy/prep requires minimum 1 thought
- Thoughts need therapyClassification field to be non-null for pattern detection
- Filter therapy thoughts via: getThoughts({ limit: 200 }) then filter client-side where therapyClassification is non-null

1. Create `vigil-pwa/src/hooks/useTherapy.ts`:
   - State: `patterns: TherapyPattern[]`, `prep: TherapyPrep | null`, `isLoadingPatterns: boolean`, `isLoadingPrep: boolean`, `error: string | null`, `therapyThoughtCount: number`
   - `analyzePatterns()` function:
     a. Fetch thoughts: `getThoughts({ limit: 200 })`
     b. Filter to therapy-classified: `thoughts.filter(t => t.therapyClassification !== null)`
     c. If fewer than 5, set error "Need at least 5 therapy-classified thoughts for pattern analysis"
     d. Map to `{ id, content, therapyClassification: t.therapyClassification!, createdAt }`
     e. Call `getTherapyPatterns(mapped)`
     f. Store patterns in state, store therapyThoughtCount
   - `generatePrep()` function:
     a. Fetch thoughts: `getThoughts({ limit: 200 })`
     b. Filter to `therapyClassification === 'bringToTherapist'`
     c. If none, set error "No thoughts classified as 'bring to therapist' found"
     d. Map to `{ id, content, createdAt }`
     e. If patterns exist, pass them to `generateTherapyPrep(mapped, patterns.map(p => ({theme, trend, confidence, description})))`
     f. Store prep in state
   - Returns `{ patterns, prep, isLoadingPatterns, isLoadingPrep, error, therapyThoughtCount, analyzePatterns, generatePrep }`
