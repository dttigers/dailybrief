## Tests

### 1. Cross-machine boot from MacBook Pro browser to iMac Vite over Tailscale
expected: On iMac, run `npm run dev` from repo root. On MacBook Pro, open `http://jamesons-imac-2:5173` in a browser. The Vigil PWA loads and renders. DevTools → Network → any `/v1/*` request has URL `http://jamesons-imac-2:5173/v1/...` (same-origin on :5173, NOT `http://localhost:3001/...`). vigil-core log on iMac shows the proxied request. Login works (auth bearer flow end-to-end).
result: passed
detail: First attempt blocked by Vite 5+ `server.allowedHosts` DNS rebinding guard — `host: true` makes Vite listen on 0.0.0.0 but the default allowlist (localhost/127.0.0.1/local network) rejected MagicDNS hostname `jamesons-imac-2`. Root-cause fixed in commit `6521373` — env-gated `allowedHosts` (default `true` = permissive, matches D-D1 "Tailscale is the perimeter"; tighter scope via `VITE_ALLOWED_HOSTS` in `.env.local`). User confirmed approval post-hotfix on 2026-04-22.
