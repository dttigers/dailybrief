## Issues Encountered

- **macOS `timeout` command missing.** The plan's acceptance-criteria smoke tests used `timeout 3 npx tsx ...` which is a GNU coreutils command not present by default on macOS. Worked around with `background-process + sleep + kill -TERM/-9 + wait` pattern — functionally equivalent, and produced the expected log lines cleanly. Not a code change, just a test-execution adjustment.
- **Shell PIPESTATUS quirk on prod FATAL probe.** First run with `2>&1 | tee /tmp/...; RC=${PIPESTATUS[0]}` produced empty `$RC` due to how zsh handles PIPESTATUS in non-interactive Bash-tool subshells. Re-ran with direct redirect (`> /tmp/...log 2>&1; RC=$?`) — exit 1 confirmed cleanly.
