Audit and fix every existing caller of `GET /thoughts` that expects all-time data. Add a typed `window?: 'all'` parameter to the PWA API client and propagate it through the four hooks that currently call `getThoughts` without date scope (`useInsights`, `useTherapy` ×2, `useProjects` ×2). D-08 requires this audit-and-fix before the rollover goes live. `useThoughts` (the hook powering the Thoughts tab) intentionally does NOT pass `window=all` — it is the intended consumer of the new default.
