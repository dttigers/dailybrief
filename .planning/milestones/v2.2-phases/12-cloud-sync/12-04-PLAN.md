<task type="auto">
  <name>Task 1: Wire SyncService into AppDelegate lifecycle</name>
  <files>Sources/DailyBriefMonitor/AppDelegate.swift</files>
  <action>
  **Add SyncService property:**
  - `private var syncService: SyncService?`
  - `private var syncTimer: Timer?`

  **In applicationDidFinishLaunching, inside the existing `do` block** (after folder watcher setup):
  1. Check `config.cloudSync.enabled`
  2. If enabled:
     - Create `CloudKitManager()`
     - Create `SyncService(cloudKit: cloudKitManager, store: thoughtStore)`
     - Store as `self.syncService = syncService`
     - Trigger initial sync: `Task { try? await syncService.sync() }`
     - Start periodic sync timer:
       ```swift
       let interval = TimeInterval(config.cloudSync.autoSyncIntervalMinutes * 60)
       syncTimer = Timer.scheduledTimer(withTimeInterval: interval, repeats: true) { [weak self] _ in
           guard let syncService = self?.syncService else { return }
           Task { try? await syncService.sync() }
       }
       ```
  3. If not enabled: skip (syncService remains nil)

  **In applicationWillTerminate:**
  - Invalidate sync timer: `syncTimer?.invalidate()`
  - Trigger final sync before quitting: `if let sync = syncService { Task { try? await sync.sync() } }`

  Note: Follow the same pattern as folderWatcher — conditional initialization based on config, stored as optional property.
  </action>
  <verify>swift build succeeds; app launches without errors when cloud_sync.enabled is false (default)</verify>
  <done>SyncService initialized on launch when enabled. Periodic timer triggers sync at configured interval. Final sync on quit. Graceful no-op when disabled.</done>
</task>
