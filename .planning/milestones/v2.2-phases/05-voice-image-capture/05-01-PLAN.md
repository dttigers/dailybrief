  2. Create VoiceCaptureService actor in JarvisCore/Services/:
     - Public actor with `AVAudioEngine` and `AVAudioFile` properties.
     - `requestMicrophoneAccess() async -> Bool` — calls `AVCaptureDevice.requestAccess(for: .audio)`.
     - `startRecording() throws` — configures AVAudioEngine input node with standard format (16kHz mono Float32 for WhisperKit compatibility), creates temp WAV file in NSTemporaryDirectory(), installs tap on input node, starts engine.
     - `stopRecording() -> URL` — stops engine, removes tap, returns the temp audio file URL.
     - `isRecording: Bool` computed property.
     - Store recording state and current file URL as private properties.
     - `VoiceCaptureError` enum: `.microphoneAccessDenied`, `.recordingFailed(String)`, `.notRecording`.

  Import AVFoundation and AVFAudio. Use `#if canImport(AVFoundation)` guard since this is macOS.
  The actor does NOT need to be @MainActor — AVAudioEngine is thread-safe.
  </action>
  <verify>swift build compiles without errors (WhisperKit resolves, VoiceCaptureService compiles)</verify>
  <done>WhisperKit in Package.resolved, VoiceCaptureService actor with start/stop recording compiles</done>

  1. Create TranscriptionService actor in JarvisCore/Services/:
     - Public actor wrapping WhisperKit.
     - `private var whisperKit: WhisperKit?` — lazy-loaded on first transcription.
     - `loadModel() async throws` — initializes WhisperKit with default model (`WhisperKit()` auto-downloads smallest model on first run). Store in `whisperKit` property. Only loads once (guard if already loaded).

     - `TranscriptionError` enum: `.modelLoadFailed(String)`, `.transcriptionFailed(String)`, `.emptyResult`.
