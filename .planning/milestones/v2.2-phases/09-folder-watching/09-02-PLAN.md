  <files>Sources/DailyBriefMonitor/AppDelegate.swift</files>
  <action>
    **Add FolderWatcherService property** to AppDelegate:
    ```
    private var folderWatcher: FolderWatcherService?
    ```

    **In `applicationDidFinishLaunching`**, after the existing service initialization block (after captureService, transcription, imageDescService are created), add:

    ```
    // Start folder watcher if enabled
    if let config = try? ConfigLoader.load(), config.folderWatching.enabled {
        let watcher = FolderWatcherService(
            config: config.folderWatching,
            transcriptionService: transcription,
            imageDescriptionService: imageDescService,
            captureService: service,
            triageService: triageService
        )
        self.folderWatcher = watcher
        Task { await watcher.start() }
    }
    ```

    Note: `service` (CaptureService) and `transcription` (TranscriptionService) are local variables in the existing do block. Place the watcher init inside that same do block, after captureService is assigned but before the panel setup — or after the panel setup, either works. The key is having access to all the service references.

    **In `applicationWillTerminate`**, add:
    ```
    if let watcher = folderWatcher {
        Task { await watcher.stop() }
    }
    ```

    **Create default folders:** When folder watching is enabled and folders don't exist, FolderWatcherService.start() already creates them. No additional work needed in AppDelegate.

    ```
        .tabItem { Label("Folders", systemImage: "folder.badge.gearshape") }
    ```

    Create the `foldersTab` computed property:
    ```
    private var foldersTab: some View {
        Form {
            Toggle("Enable Folder Watching", isOn: $viewModel.folderWatchingEnabled)

            Section("Audio Folder") {
                HStack {
                    TextField("Path", text: $viewModel.audioFolderPath)
                    Button("Choose...") {
                        if let url = showFolderPicker() {
                            viewModel.audioFolderPath = url.path.replacingOccurrences(
                                of: NSHomeDirectory(), with: "~"
                            )
                        }
                    }
                }
                Text("Drop audio files here (.wav, .mp3, .m4a, .aiff) for auto-transcription")
                    .font(.caption)
                    .foregroundStyle(.secondary)
            }

            Section("Image Folder") {
                HStack {
                    TextField("Path", text: $viewModel.imageFolderPath)
                    Button("Choose...") {
                        if let url = showFolderPicker() {
                            viewModel.imageFolderPath = url.path.replacingOccurrences(
                                of: NSHomeDirectory(), with: "~"
                            )
                        }
                    }
                }
                Text("Drop images here (.jpg, .png, .gif, .webp) for AI description")
                    .font(.caption)
                    .foregroundStyle(.secondary)
            }
        }
        .padding()
    }
    ```

    **Add `showFolderPicker()` helper** to SettingsView (private method):
    ```
    private func showFolderPicker() -> URL? {
        let panel = NSOpenPanel()
        panel.canChooseFiles = false
        panel.canChooseDirectories = true
        panel.allowsMultipleSelection = false
        panel.canCreateDirectories = true
        if panel.runModal() == .OK {
            return panel.url
        }
        return nil
    }
    ```

    **Note:** The settings frame may need height adjustment if 8 tabs don't fit. Check if the TabView handles it gracefully — if tabs wrap or get cut off, increase the frame height from 460 to 500.

    **Important:** After saving settings with folder watching newly enabled, the user must restart the monitor app for the watcher to start (since AppDelegate reads config at launch). Add a note in the UI: "Restart Jarvis to apply folder watching changes" shown when `folderWatchingEnabled` is true. Use a small Text with .foregroundStyle(.orange) below the toggle.
