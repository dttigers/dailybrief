<task type="auto">

  <files>vigil-core/src/routes/triage.ts, vigil-core/src/index.ts</files>
  <action>
    1. Create `src/routes/triage.ts`:
       - POST /triage accepting JSON body `{ content: string }`
       - Validate content is non-empty string, return 400 if missing
       - If AI client unavailable (no API key), return 503 with error message
       - Call `callClaude()` with:
         - system prompt (port exactly from Swift TriageService):
           ```
           You are a thought categorizer. Categorize the user's thought into exactly one of these categories:

           - task: actionable to-do item, something to do or buy
           - therapy: feelings, emotions, therapy questions, mental health reflections
           - idea: creative ideas, feature concepts, business ideas, "what if" thoughts
           - reflection: observations, journal entries, life reflections, gratitude
           - project: project notes, technical decisions, work-related context

           Respond with ONLY a JSON object, no other text:
           {"category": "<category>", "confidence": <0.0-1.0>}
           ```
         - userMessage: the raw thought content
         - maxTokens: 100
       - Parse response as JSON into TriageResult
       - If JSON parse fails, return 502 with "AI response parse error"
       - Return 200 with TriageResult JSON
    2. Mount in `src/index.ts`: import triage, add `app.route("/v1", triage)`
  </action>
