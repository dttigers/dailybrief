<task type="auto">
  <name>Task 1: Create POST /v1/affirmation endpoint with daily cache</name>
  <files>vigil-core/src/routes/affirmation.ts</files>
  <action>
    1. Create `src/routes/affirmation.ts`:
       - POST /affirmation accepting optional JSON body `{ recentThoughts?: string[] }`
       - Implement daily file-based cache:
         - Cache dir: `~/.cache/dailybrief/`
         - Cache file: `affirmation-YYYY-MM-DD.txt`
         - If cache exists for today, return cached text immediately
       - If no cache, call `callClaude()` with:
         - system prompt (port from Swift AIService):
           ```
           Generate a brief, warm ADHD-specific affirmation (2-3 sentences). Address themes like focus, time management, self-worth, or embracing how your brain works. Be encouraging but not patronizing. Vary the theme each day.
           ```
           If recentThoughts provided and non-empty, append:
           ```
           The user recently captured these thoughts (reference 1-2 naturally if relevant, don't force it):
           <first 5 thoughts, each truncated to 50 chars>
           ```
           Always append: `Return only the affirmation text.`
         - userMessage: `Give me today's ADHD affirmation.`
         - maxTokens: 200
       - Write response to cache file (mkdir -p cache dir, use fs.writeFileSync — fire and forget on error)
       - Return 200 with `{ affirmation: string, cached: boolean }`
       - Fallback on any AI error: return `"You are capable, you are enough, and today is full of possibility."` with `cached: false`
       - If AI client unavailable, return 503
    2. Mount in `src/index.ts`
  </action>
  <verify>cd vigil-core && npx tsc --noEmit</verify>
  <done>POST /v1/affirmation returns cached affirmation or generates new one via Claude, with daily file cache at ~/.cache/dailybrief/</done>

<task type="auto">
  <name>Task 2: Create POST /v1/insights endpoint</name>
  <files>vigil-core/src/routes/insights.ts</files>
  <action>
    1. Create `src/routes/insights.ts`:
       - POST /insights accepting JSON body:
         ```
         { thoughts: Array<{ id: number, content: string, category: string, createdAt: string }>, days?: number }
         ```
       - Validate: minimum 3 thoughts required, return 400 if fewer
       - If AI client unavailable, return 503
       - Call `callClaude()` with:
         - system prompt (port from Swift InsightService):
           ```
           You are a personal insight engine for someone with ADHD. Analyze their recent captured thoughts and surface useful patterns, connections between ideas, and actionable suggestions. Focus on being genuinely helpful, not generic. Return a JSON array of insights.
           ```
         - userMessage (port from Swift):
           ```
           Here are my captured thoughts from the last {days || 7} days:
           [{id}] ({category}, {createdAt}) {content}
           ...

           Analyze these thoughts and return a JSON array of insights. Each insight should be:
           [{"type": "pattern|connection|actionPrompt|trend", "title": "...", "message": "...", "confidence": 0.0-1.0, "related_thought_ids": []}]

           Return ONLY the JSON array, no other text.
           ```
         - maxTokens: 1024
       - Parse JSON response into Insight[] (map `related_thought_ids` → `relatedThoughtIds`)
       - Filter to confidence >= 0.5 (matching Swift behavior)
       - Return 200 with `{ insights: Insight[] }`
       - On JSON parse failure, return 502
    2. Mount in `src/index.ts`
  </action>
  <verify>cd vigil-core && npx tsc --noEmit</verify>
  <done>POST /v1/insights accepts batch of thoughts, returns filtered insights with confidence >= 0.5</done>
</task>
