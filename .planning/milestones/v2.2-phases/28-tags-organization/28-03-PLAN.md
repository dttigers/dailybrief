  **Context menu "Link to..." action:**
  1. Add "Link to..." context menu item on ThoughtRowView (SF Symbol "link"). Triggers a callback `onLinkThought: () -> Void`.
  2. In DashboardViewModel, add:
     - `var linkingThoughtId: Int64?` — the thought being linked FROM (nil when not linking)
     - `var linkSearchQuery: String = ""` — search text for finding target thought
     - `var linkSearchResults: [Thought] = []` — search results for link target picker
     - `func startLinking(thoughtId: Int64)` — sets linkingThoughtId, clears search
     - `func searchForLinkTarget(query: String)` — searches thoughts via store.search, excludes self and already-linked thoughts
     - `func createLink(targetId: Int64)` — calls store.linkThoughts, refreshes, clears linking state
     - `func removeLink(thoughtId: Int64, linkedId: Int64)` — calls store.unlinkThoughts, refreshes
     - `func fetchLinkedThoughts(thoughtId: Int64) -> [Thought]` — calls store.fetchLinkedThoughts

  **Link target picker sheet (LinkedThoughtsSheet.swift):**
  1. Create new SwiftUI view `LinkedThoughtsSheet`:
     - Presented as .sheet when linkingThoughtId != nil
     - Top section: Search TextField bound to linkSearchQuery
     - Results list: Show matching thoughts with content preview + category pill
     - Tapping a result calls createLink(targetId:) and dismisses
     - Cancel button to dismiss without linking

  **Linked thoughts indicator on ThoughtRowView:**
  1. When a thought has linked thoughts (linkCount > 0), show a small link icon + count badge near the category pill area (SF Symbol "link" with count).
  2. Add `linkCount: Int` parameter to ThoughtRowView (computed by ViewModel when loading thoughts).

  **Linked thoughts in expanded view:**
  1. When a thought is expanded (isExpanded = true) and has links, show a "Linked Thoughts" section below the content:
     - List each linked thought as a compact row (truncated content + category pill)
     - Each linked thought row has an "unlink" button (SF Symbol "link.badge.minus")

  2. Add callbacks: `onRemoveLink: (Int64) -> Void` for unlinking from expanded view.

  **DashboardViewModel loading:**
  - When loading thoughts, also compute link counts for each thought. Store as `var linkCounts: [Int64: Int] = [:]` dictionary keyed by thought ID.
