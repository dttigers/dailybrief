**ThoughtStore** — add three new methods:

1. `bulkDelete(ids: Set<Int64>) throws -> Int` — marks all matching thoughts as `pendingDeletion` in a single write transaction. Return count of rows updated. Use raw SQL: `UPDATE thoughts SET syncStatus = ? WHERE id IN (...)` for efficiency.

2. `bulkUpdateCategory(ids: Set<Int64>, category: ThoughtCategory) throws -> Int` — sets `category`, `modifiedAt = Date()`, `syncStatus = .pending` for all matching IDs in a single write transaction. If category is `.task`, also set `taskStatus = .open` for any thought that doesn't already have a taskStatus. If category is NOT `.task`, set `taskStatus = nil`. Return count updated.

3. No new method needed for bulk retriage — that's sequential AI calls handled in ViewModel.

**DashboardViewModel** — add selection mode state and bulk action methods:

State:
- `isSelectionMode: Bool = false` — toggles selection UI
- `selectedThoughtIds: Set<Int64> = []` — currently checked rows
- `isBulkProcessing: Bool = false` — shows progress during bulk retriage
- `bulkProgress: (current: Int, total: Int)? = nil` — retriage progress

Methods:
- `toggleSelectionMode()` — toggles `isSelectionMode`, clears `selectedThoughtIds` when turning off
- `toggleSelection(_ thought: Thought)` — add/remove thought.id from `selectedThoughtIds`
- `selectAll()` — adds all current `thoughts` IDs to `selectedThoughtIds`
- `deselectAll()` — clears `selectedThoughtIds`
- `bulkDelete() async` — calls `store.bulkDelete(ids:)`, then clears selection, refreshes thoughts and counts
- `bulkRetriage() async` — iterates `selectedThoughtIds`, calls `triageService.triage()` for each, updates store with result. Sets `isBulkProcessing = true` and updates `bulkProgress` as it goes. On completion, clears selection and refreshes. Skip any thought where triage throws (log warning, continue).
- `bulkRecategorize(category: ThoughtCategory) async` — calls `store.bulkUpdateCategory(ids:category:)`, clears selection, refreshes thoughts and counts.

Keep existing single-item methods unchanged.
